Images

Content Images

Fanno parte del contenuto della pagina (le immagini dei prodotti di Amazon)

Tag di riferimento: <img src="src/img/img-1.jpeg" alt="cover">

Tipologie classiche di file: JPG, PNG (trasparenti), SVG (vettoriali)

Note: Di default si settano alle dimensioni originali (500x750). In questo caso l'immagine è stata ridotta a 228x342 tramite gli attributi del tag img

Note: Anche l'elemento img è un tag con display di tipo inline-block

Note: Si può aggiungere la classe img-responsive di Bootstrap per farla adattare al contenitore padre
