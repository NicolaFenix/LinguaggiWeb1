Animations

Transitions CSS3

La proprietà transition attiva un effetto di transizione ad un elemento. Vanno specificate 2 cose:

  • La proprietà CSS che si vuole animare
  • La durata dell'effetto

CSS: transition: <proprietà> <durata> <timing-function> <delay>;

CSS: transition: width 2s ease-in-out;

Note: Si possono animare anche più proprietà insieme, con transition: width 2s, height 4s; (e con parametri diversi).

Keyframe animation CSS3

Si compone di 2 passaggi:

  1. Definizione dell'animazione
  2. Assegnazione dell'animazione

CSS: @keyframes <nome-animazione>{
        <from> {<proprietà>: <valore>;}
        <to> {<proprietà>: <valore>;}
        }
