Anonymous Functions

Se si definisce una funzione nel modo classico, questa viene immagazinata nella memoria del programma immediatamente, consumandola. E' possibile dichiarare invece delle funzioni dentro una variabile. In questo modo la funzione verrà eseguita solo nel momento in cui quella variabile viene definita. In questo modo è possibile dichiarare una funzione senza nome (ovvero anonima):


           var funzione = function(a, b){
                return a + b;
            }

            console.log(funzione(1, 3));

Funzione Map()

Map()

La funzione map() permette di eseguire una funzione per ogni singolo elemento di un array. In pratica evita di dover creare un loop che scorre l'intero array e accetta solo funzioni come parametro. E' un modo molto semplice e stringato di eseguire questo genere di operazioni.


        var numeri = [1, 2, 44, 23, 76, 13, 2, 5];

        var raddoppia = numeri.map(function(elementoArray){return elementoArray * 2}); // Come è possibile notare viene scritto tutto su una riga per essere più leggibile

        console.log(raddoppia);
