Variables

Nelle variabili posso salvare diverse informazioni, come numeri, stringhe, array. Creare una variabile è semplice, basta scrivere var, seguito dal nome della variabile e, se si vuole, uguale e il valore ad essa asseganto.


    var number = 1;
    var add = 6;
    var name = "Mentaltoy";
    var array = ["Mario", "Daniele", "Fabrizio"];

    var num;
    num = 0;

    console.log("Il numero nella variabile number è: " + number); //Lo spazio va messo nella stringa, quelli fuori dalle "" non vengono contati
    console.log("//" + num); // Ho creato prima la variabile e POI l'ho modificata con il segno =
    console.log(number + add);
    console.log(number == 1); // Sto chiedendo se una condizione è vera o no (https://www.w3schools.com/js/js_comparisons.asp)
    console.log(number == 100); // Sto chiedendo se una condizione è vera o no

    number = 100; // Cambio il valore di una variabile
    console.log(number == 100); // Ora questa condizione è vera

    number++; // Questo è un operatore aritmetico che incrementa il valore di 1 (https://www.w3schools.com/js/js_operators.asp)
    console.log(number);

    console.log(name);
    console.log(array);
    console.log(array[0]);
    console.log(array[1]);
    console.log(array[2]);
    console.log(array.length);

Notare come le variabili vengono "stampate" su quella che chiamiamo console, tramite il comando console.log(). Dove tra parentesi inserisco il valore di ciò che voglio venga restituito. E' possibile inoltre "concatenare" diversi valori potendo così differenziare i vari risultati nella console, appunto.

Funzioni

function()

Una funzione è qualcosa che vogliamo che il nostro programma faccia. Concettualmente prevede un input, che viene in qualche modo processato, ed un output, ovvero un risultato.

Una funzione fa step by step qualcosa che dobbiamo fare un maniera ripetitiva. La vera potenza delle funzioni sta nel fatto che possono essere riutilizzate quante volte vogliamo.

Una funzione si dichiara scrivendo function, seguito da spazio e dal nome della funzione e dai parametri richiesti, tra parentesi. E' seguita da due parentesi graffe che determinano la funzione stessa. Così:



    function nomeFunzione(paramentro1,parametro2){
        ...
    }

Possiamo usare quanti parametri vogliamo, questi diventaranno i "materiali" che potremo utilizzare all'interno della nostra funzione per compiere un'operazione.

Dentro le nostre parentesi scriviamo ciò che va eseguito. Alla fine della funzione (solo alla fine perchè determina un uscita immediata da essa) troviamo la parola return. Ciò che segue questa parola è ciò che vogliamo la nostra funzione "ritorni". Chiaramente utilizziamo questa parte della funzione se questa è chiamata a fare ad esempio un'operazione matematica.


    // Esempio di somma di cubi con variabili
    var a = 3;
    var b = 5;

    var aCubo = a * a * a;
    var bCubo = b * b * b;

    var somma = aCubo + bCubo;
    console.log("-------- Funzioni");
    console.log("La somma dei cubi è:" + somma);

    // Come faccio però se i valori cambiano? Meglio usare una funzione!
    function sommadiCubi(a, b) {
        var primoCubo = a * a * a;
        var secondoCubo = b * b * b;
        var sum = primoCubo + secondoCubo;
        return sum;
    }

    console.log("La somma dei cubi fatta con una funzione: " + sommadiCubi(3, 5));
    console.log("-------- Fine Funzioni");

    // Possiamo usare quanti parametri vogliamo, passando espressioni o variabili!

E' importante ricordare che le funzioni creano un nuovo scope nel programma. In pratica il programma ha uno scope globale e uno locale. Le variabili dichiarate all'esterno di una funzione vengono definite globali, e possono essere accedute all'interno delle funzioni. Al contrario, le variabili dichiarate all'interno di una funzione, hanno uno scope locale, ovvero esistono solo all'interno di una funzione, e per questo motivo non possono essere accedute dall'esterno, o da un'altra funzione:


     var x = 6;

    function aggiungi(a, b) {

        var x = a + b;
        return x;

    }

    console.log(aggiungi(9, 2));
    console.log(x); // Questo ritorna sempre 6, perchè la variabile x all'interno della funzione vale solo li

    var y = 6;

    function aggiungi2(a, b) {

        y = a + b;
        return y;

    }

    console.log(aggiungi2(9, 2));
    console.log(y); // Questo invece torna il risultato della funzione, perchè non dichiarando di nuovo la variabile all'interno, lo scope è sempre quello globale.

Array

["el1", "el2"]

Gli array sono strutture di dati, creati per non dover creare decine e decine di variabili. Possono contenere un'intera lista di variabili. Questa lista, tra l'altro, è ordinata ed elencata. In pratica ogni suo elemento ha un index.

Sono dichiarati in una variabile tramite una lista divisa da virgole e racchiusa in due parentesi quadre var amici=["Pippo", "Pluto", "Topolino"];

Nel momento stesso in cui sono inseriti, gli elementi hanno un numero ad essi associato, che parte da 0. Per accedere a un particolare elemento di un array, mi basta indicare quella variabile e un numero tra parentesi quadre es. amici[1] (In questo caso in pratica viene preso il secondo perchè si parte da 0).

Si può cambiare il valore specifico di uno dei componenti dell'array "puntandolo" e assegnandogli un nuovo valore, così: amici[1]="Paperino"


    var amici = ["Pippo", "Pluto", "Topolino"];

    console.log("Amici in principio: " + amici);

    amici[1] = "Paperino";

    console.log("Amici dopo assegnazione:" + amici);

Esistono tutta una serie di metodi (funzioni anch'esse) che permettono di modificare un array, aggiungendo e togliendo elementi.

Un method è una funzione degli array che si accede scrivendo il nome dell'array .method "punto e il nome del method".

Gli array possono contenere diversi elementi, anche array stessi. Possono inoltre contenere variabili e combinazioni di elementi diversi (numeri, variabili, stringhe)

pop()

Elimina l'ultimo elemento di un array e ritorna il valore.


    amici.pop();

    console.log("Amici dopo pop: " + amici);
push()

Aggiunge una "cella" nell'ultima posizione di un array ed aggiunge un valore. In pratica bisogna specificare cosa inserire


    amici.push("Zio Paperone");

    console.log("Amici dopo push: " + amici);;

Animazione rollover

Sono molto importanti le animazioni roll'over, perche permettono effetti molto appariscenti. In questo caso è molto interessante come applicare la stessa funzione a tutti gli elementi, passando come argomento della funzione l'evento del mouse:



    $('.arrayMouse').mouseenter(function(e) { // Questo al mouse enter

        console.log(e); // Mostra tutte le informazioni relative all'evento
        console.log(e.currentTarget); // Questo è l'elemento presente nell'attributo currentTarget


        TweenLite.to($(e.currentTarget), 1, {scaleX:1.5, scaleY:1.5, rotation:25, ease: Elastic.easeOut.config(1, 0.3) }); // Applico l'aniamzione al target specifico


    });

    $('.arrayMouse').mouseleave(function(e) { // Questo al mouse leave

        TweenLite.to($(e.currentTarget), 1, {scaleX:1, scaleY:1, rotation:0, ease: Elastic.easeOut.config(1, 0.3) }); // Applico l'aniamzione al target specifico


    });
