Transform

Transform 2d CSS3

La proprietà transform applica una trasformazione 2D ad un elemento. Permette di scalare, ruotare, spostare gli elementi in 2 dimensioni.

CSS: transform: <funzione>;

CSS: transform: translate(20px,30px) scaleY(0.5);

Transform 3d CSS3

La proprietà transform applica una trasformazione 3D ad un elemento. Permette di scalare, ruotare, spostare gli elementi in 3 dimensioni.

CSS: transform: <funzione>;

CSS: transform: rotateZ(35deg) rotateY(45deg);

Transform origin CSS3

La proprietà transform-origin modifica il punto di inserimento dell'elemento, e quindi il punto di applicazione delle trasformazioni.

CSS: transform-origin: <valoreX valoreY>;

Skew
0 0
50% 50%

Importante per il punto di ancoraggio che, nel caso di trasform origin di default (50% 50%), è come se fosse in un punto indeterminato nello spazio. Nel caso di 0 0, invece, coincide con il punto di applicazione della trasformazione e quindi è ben identificabile.

Rotate
0 0
50% 50%

Anche in questo caso il punto di inserimento fa la differenza. Se serve muovere l'oggetto come se fosse la lancetta di un orologio conviene l'origin modificata (0 0). Per il movimento di una ruota invece quello di default (50% 50%) è perfetto.

Scale
0 0
50% 50%

Nel caso della scala, la transform-origin modificata permette di realizzare una barra di caricamento che si espande dalla sinistra verso la destra. In altri casi invece potrebbe essere più conveniente che l'elemento si espanda in maniera proporzionale.

Transform + Animation

Si può combinare una trasformazione SSR e un'animazione per ottenere un effetto interessante.

CSS: transform:matrix(0.87,0.50,-0.69,0.59,0,0);
